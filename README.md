# AutoRent | Modern Car Rental Landing Page

A premium and high-performance car rental landing page built with **React 19**, **Vite**, and **Tailwind CSS v4**.  
This project focuses on a sleek UI, seamless navigation, and a robust architecture for a modern transportation service experience.

## Features

- **Responsive Design:** Fully optimized for mobile, tablet, and desktop viewports.
- **Dynamic Routing:** Smooth page transitions using React Router 7.
- **Modern UI Components:** Integrated with Material UI and Lucide React for a professional look.
- **Engaging Animations:** Scroll-based reveal effects using ScrollReveal.js.
- **Interactive Sections:** Featured cars showcase, video integration, and trust-building testimonials/features.
- **Optimized Performance:** Lightning-fast build and reload times thanks to Vite and Tailwind v4.

## Tech Stack

- **React 19** (Latest Features)
- **Vite** (Next Generation Frontend Tooling)
- **Tailwind CSS v4** (Modern Utility-First CSS)
- **React Router 7** (Client-side Routing)
- **ScrollReveal** (Scroll Animations)
- **Lucide React & MUI Icons** (Iconography)
- **Material UI** (Component Library)

## Animations

The project utilizes **ScrollReveal** to create an interactive "reveal-on-scroll" experience, making the interface feel alive as the user explores the fleet.

Included motion features:

- **Distance-based reveals:** Elements slide into view from different directions.
- **Staggered loading:** Grid items (like car cards) appear one after another.
- **Clean Transitions:** Subtle opacity fades for professional-grade UX.

## Pages / Sections

The landing page includes:

- **Navigation:** Responsive header with route links.
- **Hero Section:** High-impact call-to-action with search intent.
- **Featured Cars:** A dynamic showcase of available vehicles.
- **Video Section:** Immersive brand storytelling through video.
- **Features/Benefits:** Grid of service advantages (Insurance, 24/7 support, etc.).
- **Auth Pages:** Dedicated Login and Register layouts.
- **Footer:** Multi-column navigation and social links.
